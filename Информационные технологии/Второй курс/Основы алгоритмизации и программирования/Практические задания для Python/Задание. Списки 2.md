## Условие

Пусть дан список-матрица, который содержит три списка:

```python
mat = [
    [10, 20, 30],
    [40, 50, 60],
    [70, 80, 80]
]
```

Выведите элементы матрицы с помощью циклов, чтобы получился консольный вывод как в следующем примере:

**Пример 1:**

```

```
